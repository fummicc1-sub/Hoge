### 配列とは
同じ種類のデータ(値)をまとめたもの。

### 配列の使い方
```swift
var 変数の名前: [配列の要素の型] = [要素1, 要素2, 要素3]
```

---

### print
コンソールという場所に指定した変数の値が表示されます。

**例**
```swift

var number = 10

print(number)

```
